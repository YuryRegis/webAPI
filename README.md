# webAPI
WebAPI utilizando Node + Typescript


# Requisitos

    - Devolva uma lista de registros
    - Opcional: permitir filtros
    - Adicione um registro a lista
    - Remove um registro da lista
    - Atualiza um registro da lista


# Build

``` $ yarn install ```
``` $ yarn build ```

# Run locally (dev)

``` $ yarn dev ```
